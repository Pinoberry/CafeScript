# CaféScript

Tienda en línea de café premium chileno. Desarrollada con React, TypeScript y Tailwind CSS.

## Características

- Catálogo de productos
- Carrito de compras
- Filtrado por región
- Diseño responsivo

## Instalación

```bash
npm install
npm run dev
```
